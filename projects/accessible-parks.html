Accessible Parks Search for Iowa

Overview

A searchable web application that helps users locate parks and outdoor recreation areas based on accessibility features.

Problem

Accessibility information for parks is often difficult to locate across multiple websites and resources.

Solution

Create a centralized search experience where users can discover accessible outdoor spaces throughout Iowa.

Key Features

  • Search parks by accessibility feature
  • Filterable listings of parks and recreation areas
  • Centralized accessibility information in one location
  • Keyboard-accessible search and results interface

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Generative AI

What I Learned

  • Accessibility-first design
  • Information architecture
  • Search experience design
  • User-centered design

Future Enhancements

  • Integration with real-time trail condition updates
  • User-submitted accessibility reviews and photos
  • Offline access for users in areas with limited connectivity
  • Expanded coverage beyond Iowa state parks
