Youth Sports Team Manager

Overview

A team management platform that helps youth baseball coaches, players, and parents manage schedules, attendance, and communication.

Problem

Coaches often manage team information using spreadsheets, emails, texts, and multiple disconnected tools.

Solution

Create a centralized application where coaches can manage teams and parents can track attendance and player status.

Key Features

  • Coach login
  • Parent login
  • Team creation
  • Player roster management
  • Parent contact management
  • Practice scheduling
  • Game scheduling
  • Attendance tracking
  • Injury reporting
  • Absence reporting

Technologies Used

  • HTML
  • CSS
  • JavaScript
  • Generative AI

What I Learned

  • User roles and permissions
  • Dashboard design
  • Scheduling workflows
  • Information architecture
  • User-centered design

Future Enhancements

  • Push notifications for schedule changes and announcements
  • Team photo galleries and season highlights
  • Integration with league registration systems
  • Mobile app for on-the-field updates
